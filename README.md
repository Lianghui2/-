                                           实验室预约系统的设计与实现
功能：1，信息输入，建立实验室基本信息，实验信息等基础数据；
      2，开放信息公布；设置开放信息并公布；
      3，预约：学生设置预约实验室的名称、时段、实验条件等；
      4，预约审核；
开发环境：eclipse，tomcat，mysql。

预约流程：查询信息------提交申请-----审核-------进行实验------审核实验------生成实验记录

预约形式：个人预约；集体预约     开放形式：机位；实验室。

时间段：日期（周次），1-10节课

注册信息：姓名，学号

违约处理

用户类型：管理员，学生，教师
